# PyPassword Generator

Welcome to **PyPassword Generator**, a simple Python tool designed to generate secure and customizable passwords.

## How It Works

This password generator allows you to create a random password consisting of letters, numbers, and symbols based on your preferences for password complexity. You have full control over the number of characters, symbols, and numbers in your password.

### Features

- Choose the number of letters, symbols, and numbers for your password.
- Generates a randomized password based on your inputs.
- Option to shuffle characters to enhance password security.

### Levels

- **Easy Level**: (Commented out in the code)
    - Simply adds the specified number of letters, symbols, and numbers in sequence.
  
- **Hard Level**: 
    - Generates a list of characters (letters, symbols, numbers) and shuffles them to create a more secure, randomized password.

## How to Run the Program

1. Clone or download the code file to your local machine.
2. Make sure Python 3.x is installed.
3. Open the terminal and run the following command:
